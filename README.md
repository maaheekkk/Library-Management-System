# 📚 Library Management System (C++)

A menu-driven Library Management System implemented using a Singly Linked List.

## Features
- Add Book
- Display All Books
- Search Book by ID
- Search Book by Title
- Delete Book
- Issue Book
- Return Book

## Data Structure Used
- Singly Linked List

## Concepts Used
- Structures
- Pointers
- Dynamic Memory Allocation
- Functions
- Linked List Operations

## Sample Menu

```
1. Add Book
2. Display Books
3. Search by ID
4. Search by Title
5. Delete Book
6. Issue Book
7. Return Book
8. Exit
```

./library
